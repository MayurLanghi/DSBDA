This is the classic Apache Hadoop MapReduce WordCount program. 

Its purpose is:

```text id="w5m56m"
Count how many times each word appears in a file
```

Example:

Input:

```text id="q4aw9d"
Hello Hadoop Hello World
```

Output:

```text id="qmfeq2"
hadoop 1
hello 2
world 1
```

---

# Overall Working

The program works in 3 parts:

```text id="t9hvjp"
Input File
    ↓
Mapper
    ↓
Shuffle & Sort
    ↓
Reducer
    ↓
Final Output
```

# 3. Mapper Class

```java id="d4qtq7"
public static class TokenizerMapper
```



Mapper reads input line-by-line.

---

## Mapper Type Declaration

```java id="gnmzdu"
extends Mapper<Object, Text, Text, IntWritable>
```



Format:

```text id="twclhv"
Mapper<InputKey, InputValue, OutputKey, OutputValue>
```

Meaning:

| Type          | Meaning                 |
| ------------- | ----------------------- |
| `Object`      | Input key (line number) |
| `Text`        | Input line              |
| `Text`        | Output word             |
| `IntWritable` | Output count            |

# Create Constant One

```java id="kvn6vg"
private final static IntWritable one = new IntWritable(1);
```

Every word occurrence contributes:

# Create Word Variable

```java id="d9xk5d"
private Text word = new Text();
```



Stores each word temporarily.

---

# Mapper Function

```java id="6lb9bf"
public void map(Object key, Text value, Context context)
```



This runs for every line.

---

# Convert Line Into Words

```java id="70v6g8"
StringTokenizer itr = new StringTokenizer(value.toString());
```



Example line:

```text id="k67u8i"
Hello Hadoop Hello World
```

Tokenizer splits into:

```text id="g3jlwm"
Hello
Hadoop
Hello
World
```

---

# Loop Through Words

```java id="o87bd2"
while (itr.hasMoreTokens())
```



Processes each word one by one.

---

# Extract Word

```java id="qeh6iw"
word.set(itr.nextToken().toLowerCase());
```



Gets next word and converts to lowercase.

Example:

```text id="x5f6vp"
Hello → hello
```

This avoids separate counts for:

```text id="ddwckm"
Hello
HELLO
hello
```

---

# Emit Mapper Output


# Shuffle and Sort Phase

Automatic Hadoop step.

Groups same keys.

Before shuffle:

```text id="pkrf6w"

hadoop → 1
hello → 1
world → 1
```

After shuffle:

```text id="jdxruj"
hello → [1,1]
hadoop → [1]
world → [1]
```

# Result Variable

---

# Reduce Method

```java id="ekmjlwm"

                   Context context)
```



Example input:

```text id="7qg5wc"

```

---

# Sum Values

```java id="p2h7ya"
int sum = 0;
```



---

# Add All Counts

```java id="rfngrl"
for (IntWritable val : values) {
    sum += val.get();
}
```



Example:

```text id="v4owt7"

```

---

# Store Result

```java id="2lkl08"

```



---

# Write Final Output

```java id="5a0wj7"
context.write(key, result);
```



Output:

```text id="t8e0az"

```

# Viva Answer

You can say:

> This Hadoop MapReduce program counts word frequency in a text file.
>
> The Mapper tokenizes each line into words and emits `(word, 1)` pairs.
>
> Hadoop automatically groups identical words during shuffle and sort.
>
> The Reducer sums all counts for each word and outputs the final frequency.
