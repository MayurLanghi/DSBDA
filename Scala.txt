# How This Works

## `(2 until n)`

Creates numbers from:

```text id="hvw01u"
2 to n-1
```

Example for `n = 7`:

```text id="1svptl"
2,3,4,5,6
```

---

## `n % _ == 0`

Checks divisibility.

Example:

```text id="1r2dsh"
7 % 2
7 % 3
7 % 4
...
```

If remainder becomes `0`, then number is not prime.

---

## `exists(...)`

Checks whether ANY number divides `n`.

Example:

For `8`:

```text id="8n69z7"
8 % 2 == 0
```

So:

```text id="2d53i0"
exists = true
```

meaning:

```text id="l4a5q7"
8 is NOT prime
```

---

## `!`

Negation operator.

If divisor exists:

```text id="mdx2fy"
true → false
```

If divisor does not exist:

```text id="quj4th"
false → true
```

So:

| Number | Result    |
| ------ | --------- |
| 7      | Prime     |
| 8      | Not prime |

# Difference Between Hadoop and Spark

| Hadoop                | Spark                   |
| --------------------- | ----------------------- |
| Disk-based processing | Memory-based processing |
| Slower                | Faster                  |
| Uses MapReduce        | Uses RDD/DataFrames     |
| Older                 | More modern             |

Both are Apache projects.
